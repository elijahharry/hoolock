# Security Policy

## Supported Versions

Supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 2.2.0   | :white_check_mark: |
| < 2.2.0 | :x:                |

You can also report vulnerabilities in prior versions, I may or may not fix them depending on the severity and version age.
